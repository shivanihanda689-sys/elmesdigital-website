# Elmes Digital Website

Production-ready React + Vite + Tailwind starter for the Elmes Digital marketing website.

## Quick start

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

The production files will be created in `dist/`.

## Deploy

### Vercel
1. Create a new Vercel project
2. Upload this folder or connect your Git repo
3. Framework preset: **Vite**
4. Build command: `npm run build`
5. Output directory: `dist`

### Netlify
1. Create a new site from Git or drag-and-drop
2. Build command: `npm run build`
3. Publish directory: `dist`

## Notes
- Portfolio screenshots use Microlink URLs. If any preview fails, the site gracefully shows a fallback card.
- The “Book consultation” modal is front-end only. Hook the form up to your email/CRM/Calendly next.
